# To-Do-List

To-Do List — это веб-приложение для создания и управления списком задач. Пользователь может добавлять новые задачи, отмечать их как выполненные и удалять. Проект разработан с использованием HTML, CSS и JavaScript и помогает на практике освоить работу с DOM и пользовательскими событиями.
